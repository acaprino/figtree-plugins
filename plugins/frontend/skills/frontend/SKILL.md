# Frontend Web Development

Reference knowledge for web-specific frontend work -- CSS, UX, UI patterns, layouts, and flows.

## CSS Architecture & Modern Features

Deep reference for modern CSS including Container Queries, View Transitions, Scroll-driven Animations, Anchor Positioning, Masonry, `@scope`, Cascade Layers, `oklch()`, `clamp()`, and architecture methodologies (BEM, CSS Modules, Cascade Layers).

- **CSS patterns and examples**: Read [references/css-patterns.md](references/css-patterns.md)
- **Modern CSS talk notes (Cascadia 2025)**: Read [references/argyle-cacadia-2025-deck.md](references/argyle-cacadia-2025-deck.md)

## UX Patterns

Decision rules for onboarding, trust/social proof, persuasion/conversion, and cognitive load patterns. When to use each pattern, what makes it work, and what kills it.

- **UX pattern decision guide**: Read [references/ux-patterns.md](references/ux-patterns.md)

## UI Component Patterns

Decision rules for choosing the right component per context -- cards vs list vs table, navigation patterns, pagination vs infinite scroll, page archetypes (dashboard, product, pricing, wizard).

- **UI pattern selection guide**: Read [references/ui-pattern-guide.md](references/ui-pattern-guide.md)

## Layout Patterns

CSS layout patterns for common page structures -- Holy Grail, Full-Bleed, Split Screen, Bento Grid, Sidebar+Main, Masonry, Editorial Asymmetry, and more.

- **Layout patterns library**: Read [references/layout-patterns.md](references/layout-patterns.md)

## Flow & Onboarding Patterns

Spatial structure for guided, sequential, and focused-attention UIs -- step indicators, quiz layouts, coachmark overlays, notification positioning, paywalls, completeness meters.

- **Flow and onboarding patterns**: Read [references/flow-patterns.md](references/flow-patterns.md)
